TSQL Queries

Set of Queries developed in a group setting with the documantation used to keep track of group members progress.

About the Project

Overview

The project's goal was to take sample databases used with SQL server managment software, such as the AdventureWorks, NorthWinds databases. Using sql queries to satisfy group created propositons, on various databases loaded on local college server.

Technologies Used/Involved

  • Transact SQL(T-SQL) Language
  • SQL Server 2017 by Microsoft Download
  • Microsoft JDBC Driver Download
  • SQL Server Management Studio Download

GitHub

Github repo location of group files that were submited such as the queries of the individual group members, the materials used to keep track of the current progress of the group. Also included the links to the backup of the databases used provided by Microsoft.
