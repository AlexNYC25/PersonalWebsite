Blinkt! Morse

A GUI tool to convert a string into Morse with Blinkt HAT.

About the Project

Overview

The projects goal was to create a GUI tool that would process a string and pass the appropriate actions to display the morse on the Blinkt HAT. (Currently under construction)

Technologies Used/Involved

  • Python 3.7 Language Documantation
  • Python Tkinter GUI library Documantation
  • Blinkt library Download
